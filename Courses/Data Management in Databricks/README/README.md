# Data Management in Databricks

**Level:** Basic

**Updated:** Nov 2025

**Duration:** 3 hr | Videos: 10 | Exercises: 31 | XP: 2350

## Description

Covers Delta Lake fundamentals including ACID transactions, schema enforcement, time travel, managed vs unmanaged tables, views and temp views, Data Explorer, table governance, and secure handling of PII.

## Modules

- 1 - Introduction to Delta Lake (10 lessons)
- 2 - Working with Tables in Databricks (6 lessons)
- 3 - Data Exploration and Security (6 lessons)
