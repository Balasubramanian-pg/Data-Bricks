# Introduction to Databricks SQL

**Level:** Intermediate

**Updated:** Feb 2026

## Description

Covers Databricks SQL as a scalable lakehouse data warehouse solution. Includes medallion architecture workflows, ANSI SQL analytics, visualizations and dashboards, window functions, and merge patterns.
